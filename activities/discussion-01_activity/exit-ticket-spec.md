# Discussion 01 — Exit Ticket (PrairieLearn spec)

This file specifies the Lab 01 exit ticket. The actual question is authored in the external PrairieLearn course; this spec documents its content and intent so it can be ported and kept in sync.

## Prompt

> What is one responsibility you have when discussing sensitive AI ethics topics in this course?

## Format

- **Type:** short free-response (a few sentences).
- **Grading:** completion / participation — not correctness. Any thoughtful, on-topic answer earns full credit.
- **Timing:** completed individually and asynchronously, after Lab 01.
- **Length:** 1–3 sentences.

## Purpose

Captures each student's personal takeaway from building the class code of conduct and reinforces the lab's **Reflect** learning objective, without spending in-class time.

## Porting notes

- Author the question in the PrairieLearn course (e.g. a free-text / `pl-string-input` question with completion credit).
- Once it exists, register the link by adding this row to `data/additional-resources.csv`:
  `discussion-01-exit-ticket,prairielearn,<url>`
- Add a matching `data/schedule.csv` row only if it should appear as its own item on the course schedule.
